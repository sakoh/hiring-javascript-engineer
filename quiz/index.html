  6. What is the DOM? And the BOM? What's the difference? Give us a one-line example of using each one. The DOM(Document Object Model) is a representation of the markup in the usually HTML, and sometimes XML or XHTML, file. BOM(Browser Object Model) deals more with the other objects in the browser that are not related to the markup such as the url, the location, and history.
